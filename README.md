# Django-banner

Banners base django.
Detailed documentation is in the "docs" directory

## Quick start

1. pip install adminsortable2
2. Add "banner" to your INSTALLED_APPS setting like this:
    ```
    INSTALLED_APPS = (
        ...
        'adminsortable2',
        'banner',
    )
    ```
3. Run `python manage.py migrate` to create the banner models.
